🔑 Create First Admin

To promote an existing user to admin, run this command in your backend directory: 

python -c "
from app.database import SessionLocal
from app.models import User
db = SessionLocal()
u = db.query(User).filter(User.email=='your@email.com').first()
u.is_admin = True
db.commit()
print('Done -', u.email, 'is now admin')
db.close()
